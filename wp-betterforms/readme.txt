Blazing-fast WordPress form builder with visual styler and AJAX submissions.

== Description ==

WP Better Forms is a modern form builder that pairs a React-powered admin experience with a lightweight frontend renderer. Features include conditional logic, scoped design tokens, AJAX submissions, and a built-in email queue with failure monitoring.

== Installation ==

1. Upload the `wp-betterforms` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Open **Better Forms** in the dashboard to create your first form.

== Frequently Asked Questions ==

= Does it require Gutenberg? =

No. A block is registered when Gutenberg is available, but the main builder lives in a dedicated admin page.
